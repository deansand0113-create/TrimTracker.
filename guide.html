Project Overview

How TrimTracker is organized.

This page explains the site structure, design choices, and interactive features used in the final submission.

Purpose

TrimTracker helps users understand F-150 trims through clear research, visuals, and decision tools.

Audience

The target user is someone shopping for a truck or comparing F-150 trims for work, towing, daily use, or off-road driving.

Design

The final design uses dark colors, red accents, metallic cards, large images, animation, and responsive layout.

Interactivity

The project includes a build configurator, towing calculator, trim comparison tool, terrain selector, gallery filters, and quiz.
